# Contributing

Thank you for your interest in contributing to OrmKit!

## Ways to Contribute

- **Report bugs** - Open an issue with reproduction steps
- **Suggest features** - Discuss ideas in GitHub Discussions
- **Improve docs** - Fix typos, add examples, clarify explanations
- **Submit code** - Bug fixes, new features, performance improvements

## Quick Links

- [Development Setup](development.md) - Get your environment ready
- [Architecture Overview](architecture.md) - Understand the codebase

## Code of Conduct

Be respectful and constructive. We're all here to build great software together.

## Getting Help

- **Questions**: Open a GitHub Discussion
- **Bugs**: Open a GitHub Issue
- **Security**: Email security@ormkit.dev (do not open public issues)
