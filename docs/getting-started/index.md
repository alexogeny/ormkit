# Getting Started

Welcome to OrmKit! This guide will help you get up and running quickly.

## What is OrmKit?

OrmKit is a Python ORM that combines:

- **SQLAlchemy-style API** - Familiar declarative models and query patterns
- **Rust performance** - Core operations happen in compiled Rust code
- **Built-in drivers** - No need for asyncpg or aiosqlite dependencies
